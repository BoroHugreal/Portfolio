CARTE 02 Mission Data Viz Full-stack · MVC

Plateforme d'analyse météorologique

Centraliser, exploiter et visualiser des données météorologiques issues de sources ouvertes, avec intégration automatique des nouvelles données et fonctionnalités de suivi pour les utilisateurs.

PHPMySQLJavaScriptHTML/CSSAPI RESTArchitecture MVC

↑ Démo live des fonctionnalités du projet (données en direct, graphiques) — plutôt qu'une capture statique. Source temps réel : Open-Meteo.

Contexte

Les données météo ouvertes (réseau SYNOP) sont riches mais brutes. L'enjeu : les centraliser, les rafraîchir automatiquement et offrir une lecture claire (graphiques, cartes, tableaux), tout en gérant des comptes utilisateurs avec leurs propres « météothèques ».

Réalisation

  • Architecture MVC en PHP, récupération automatique des données depuis l'API SYNOP, intégration sans intervention manuelle.
  • Base de données structurée séparant données météo dynamiques et données métier (comptes utilisateurs, météothèques personnalisées, journaux d'activité admin).
  • Interface HTML/CSS/JS interactive : graphiques, cartes météorologiques (cartes thermiques), tableaux de données.
